Assist-Star is a swipe-based community help platform that connects people who need assistance with nearby volunteers in a fast and intuitive way.

Inspired by modern matching interfaces, the platform focuses on simplicity, accessibility, and real-world usefulness for local communities.

---

## Problem Statement

People often struggle to find timely and reliable help for everyday needs within their local community. Existing solutions are either too fragmented, slow, or difficult to use, especially in urgent or small-scale situations.

---

## Solution

Assist-Star provides a simple role-based system where users can either request help or volunteer to assist others. Through a swipe-based interface, users can quickly browse nearby matches and connect instantly, making community support faster and more approachable.

---

## Key Features

- User authentication using Firebase
- Role-based flow for help seekers and volunteers
- Swipe-based matching interface
- Location-aware matching (demo-based)
- Clean and mobile-friendly UI
- Community-focused interaction model

---

## Tech Stack

- React + Vite
- Firebase Authentication
- Tailwind CSS
- Google Cloud (Firebase services)

---

## How It Works

1. Users sign up and choose a role: **Help Seeker** or **Volunteer**
2. Help seekers describe the assistance they need
3. Volunteers browse nearby requests using swipe actions
4. On a match, users can directly connect and coordinate

---

## Use Case Examples

- Elderly individuals needing assistance with daily tasks
- Students offering tutoring or technical help
- Local volunteers responding to short-notice requests
- Community members supporting each other during emergencies
